# MyBlog-codu
My blog and Portfolio site  
HTML, CSS, 바닐라js을 이용해 내 블로그와 포트폴리오를 만들어보자!  

<br>

## 목적
- HTML, CSS의 완벽한 이해와 활용을 위해서
- 나만의 웹사이트를 내가 직접 만들어 보고 싶어서

<br>

## 기술 스택
- Front-end : HTML5, CSS, javascript  
- Back-end : Node.js(Express), Firebase

<br>
